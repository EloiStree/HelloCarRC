# Communication 

In this workshop we will learn to communicate with UART that are small text send by USB or by Bluetooth in our case.  
To do that you need a TTL if you want to use USB and a HC05 if you want to use Bluetooth.  
I tip to use both and the workshop show how to use both.  


## TTL

This device will allows you to communicate with any micro controller at high speed and without any lost of on the message transport.
